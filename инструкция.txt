Для запуска программы необходимо:
1. Установить архив Aflyatunova_Aigul.zip и распаковать его
2. Открыть папку с помощью PyCharm
3. В терминале PyCharm прописать следующие команды:
    cd veb
    python manage.py runserver
(для windows)
    cd veb
    python3 manage.py runserver
(для Linux или MacOS)
4. Нажать на появившуюся в терминале локальную ссылку 

Для просмотра базы данных необходимо зайти в панель администратора:
1. Добавить /admin после локальной ссылки в браузерной строке 
2. Авторизоваться под логином и паролем admin password
3. Таблица Events является БД по которой обучалась модель искусственного интелелкта
